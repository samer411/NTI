# NTI Full Stack Internship Journey

## Overview

This repository documents my journey through the Full Stack Open Source Track at the *National Telecommunication Institute (NTI)*.

Throughout this internship, I focused on building a strong foundation in web development by applying core concepts through hands-on tasks and mini-projects. The work included developing structured web pages,
implementing forms and data handling, and simulating real-world interfaces such as portfolio pages, social platforms, and content-driven websites.

This repository reflects my approach to learning by building, with an emphasis on writing clean, structured, and scalable code as a foundation for more advanced frontend and full stack development.
